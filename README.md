# Titanik ML Challenge

## Бинарная классификация таргета выжил/не выжил для пассажиров Титаника.

## Предобработка данных:
1) заполнение модой / удаление пропусков
2) Кодирование категориальных признаков

 ## EDA
1) Смотрим распределение в классах разных по выживаемости, строим charts.
2) Построены распределения людей по классам, цене билета.
3) Построены boxplots для выборик выживших и нет по различным признакам.
4) отбор признаков для обучения, построена корреляционная матрица признаков
   Отобраны признаки features = ['Pclass', 'sex_encoded', 'Fare']
5) Проверка с помощью VIF на независимость признаков.

## Построение модели
1) StandardScaler - нормируем числовые признаки нормальным распределением в область [0, 1]
2) Обучаем модель Логистическую регрессию LogisticRegression из sklearn.
3) Получены метрики для выжившего класса(1): precision= 0.74, recall=0.69, f1-score=0.71
Для не выжившего класса(0): precision= 0.81, recall=0.85, f1-score=0.83
4) Построена кривая ROC-AUC. AUC = 0.84
5) Реализована кросс-валидация (5 частей)
6) Реализован гиперпараметров через GridSearchCV
